# 🍕 Pizza Website README

## 📌 Overview

This is an interactive and modern pizza website created using web technologies to provide users with a visually attractive and smooth experience. The website includes high-quality pizza images, engaging animations, responsive design, and user-friendly features that make browsing enjoyable.

The project is designed for food businesses, restaurant showcases, or portfolio purposes. It focuses on combining beautiful UI design with interactive functionality.

---

## ✨ Features

* 🍕 Attractive pizza gallery with high-quality images
* 🎨 Modern and responsive user interface
* ⚡ Smooth animations and interactive effects
* 📱 Mobile-friendly design for all screen sizes
* 🛒 Interactive menu section
* 🔍 Easy navigation and clean layout
* 🌙 Stylish color combinations and visual effects
* 📧 Contact / Order section included
* 🚀 Fast loading and optimized performance
